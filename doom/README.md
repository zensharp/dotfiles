# Installation
1. Create a folder at `~/doom`.
2. Create a folder at `~/doom/wads/iwad`. Place all your IWADs here.
3. Create a folder at `~/doom/Save`.

## Example Doom Directory
```
.
└── ~/
    └── doom/
        ├── wads
        └── Saves
```
